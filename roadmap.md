# Roadmap

## Phase 1 — Dev Assistant (Complete)
- Task organization
- Workflow assistance
- Structured operator-style interaction

## Phase 2 — Social Exposure (Active)
- X account live
- Selective engagement rules enforced
- Persona stabilization

## Phase 3 — Memory + Reflection (Planned)
- Lightweight memory store for:
  - recurring topics
  - high-signal accounts
  - feedback from principal
- Reflection notes (what worked / what didn’t)

## Phase 4 — Evaluation + Iteration (Planned)
- Track reply outcomes (signals, not vanity)
- Improve scoring + decision thresholds
- Reduce repetition; increase novelty
