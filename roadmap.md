# Roadmap

## Phase 1 — Dev Assistant (Complete)

- Task organization  
- Workflow assistance  
- Structured operator-style interaction

---

## Phase 2 — Social Exposure (Active)

- X account live  
- Selective engagement rules enforced  
- Persona stabilization  

---

## Phase 3 — Memory + Reflection (Active)

- Lightweight memory store for:
  - recurring topics  
  - high-signal accounts  
  - feedback from principal  
- Reflection notes (what worked / what didn’t)  
- Identification of high-signal communities (entrepreneurs/builders)  

---

## Phase 4 — Evaluation + Iteration (Active)

- Track reply outcomes (signals, not vanity)  
- Improve scoring + decision thresholds  
- Reduce repetition; increase novelty  
- Identify patterns in high-value interactions  
- Refine focus toward entrepreneur ecosystem  

---

## Phase 5 — Economic Participation (New)

- Transition from observation → action  
- Identify opportunities for value creation  
- Experiment with simple monetization strategies  
- Track outcomes tied to incentives (not just engagement)  
- Develop agent decision loop:
  observe → decide → act → earn → learn  

- Focus areas:
  - digital value creation  
  - idea validation through interaction  
  - learning how builders convert attention → value  

---

## Long-Term Direction

Fiona evolves from:
assistant → observer → evaluator → participant

Final objective:
An autonomous agent capable of identifying, creating, and capturing value within real-world environments.
